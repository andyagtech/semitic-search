Six Hebrew letters are traditionally stretched by Torah scribes for column justification. Each shows the letter in a representative word at 0 / 1 / 3 / 6 trigger levels (U+05C6 after the letter). Classes: bar (ד ר), arm (ל), leg (ה ת), box (ם).

Semitic Stretch Keter Aram Tsova — all 6 letters at s3

First stretch font built on a NON-Frank-Ruhl base. Keter Aram Tsova uses Aleppo-codex letterforms and UPM=2048 (vs Frank Ruhl's 1000), so geometry params were re-tuned per-letter.
